## Technology stack
- Framework: Spring boot 3.5.6
- Modules spring: Spring Data, Spring WEB, Spring Security
- Build: Gradle
- ORM: Hibernate
- DB: postgres
- DB for tests: h2
- Swagger: springdoc-openapi
- Front: thymeleaf
- Migration: liquibase
- Utils: Logback, Lombok
